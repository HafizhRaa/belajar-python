# Belajar Python 🐍

Repositori ini berisi Proyek-proyek hasil belajar python dari 0.

## Proyek

### 1. Kalkulator
Kalkulator sederhana dengan 4 operasi matematika (tambah, kurang, kali, bagi).
- Fitur: Loop sampai user keluar, handle bagi dengan 0

### 2. To-do List
Aplikasi to-do list sederhana menggunakan list python.
- Fitur: tambah, lihat dan hapus tugas

### 3. Web Scraper
Program untuk mengambil quotes dari internet secara otomatis.
- Library: requests, BeautifulSoup

### 4. To-do List + Database
Upgrade dari to-do list biasa, data tersimpan permanen menggunakan SQLite.
- Fitur: tambah, lihat, tandai selesai, hapus tugas

## Tech Stack
- Python 3.14
- SQLite3
- Requests & BeautifulSoup4
